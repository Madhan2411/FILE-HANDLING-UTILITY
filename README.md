## **Project Overview**

The **File Handling Utility** is a Java-based program designed to perform three core operations on text files:

### **1. Write to a File**

Creates a new file and writes initial content to it.

### **2. Read from a File**

Reads and displays the file content line by line.

### **3. Modify (Append) File**

Adds new content to the existing file without deleting the original text.

These operations help understand how Java interacts with the file system using classes like:

* `FileWriter`
* `FileReader`
* `BufferedReader`
* `IOException`
